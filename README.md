# teste-tecnico-jukebox

Teste técnico para vaga de desenvolvedor full stack na Jukebox

Acesse primeiramente o README.md da api e posteriormente do front-end para o passa a passo de como configurar e rodar a aplicação.

# Stack utilizada

## Back-End (API)
- PHP 8.3.1
- MySQL 8
- Laravel 10
- Firebase: kreait/laravel-firebase 5.6
- Laravel Sail 8.3
- Docker 24.0.7, build afdd53b
- Docker Compose docker-compose version 1.29.2, build unknown
- PHPCS

## Front-End
- Vue.js 3.2.13
- Vue Router 4.2.5
- firebase 10.7.1
- Bootstrap 5.3.2
- ESLint 8.56

# Padrões

- Rest

## Arquitetura

- MVC

- Dependency injection;
- Service container;
- Traits.
